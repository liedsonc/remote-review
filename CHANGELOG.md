# Changelog

All notable changes to this project are documented in this file. Entries are generated from
conventional commits via [git-cliff](https://git-cliff.org) on every release tag.

## [0.1.0]

Initial release.

- Tunneled diff review server with a random per-session token.
- Blocking CLI contract: prints inline comments to stdout on submit.
- Diff targets: working tree, staged, HEAD, a ref, or a ref pair.
- `skills/remote-review` skill for Claude Code dispatch sessions.
